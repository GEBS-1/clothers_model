# Инструкция: Как перехватить запрос из UI Space

## Шаг 1: Открой Space в браузере
1. Открой `https://levihsu-ootdiffusion.hf.space` (или любой другой рабочий Space)
2. Открой DevTools (`F12`)
3. Перейди на вкладку **Network**
4. Включи фильтр **Fetch/XHR**

## Шаг 2: Выполни действие в UI
1. Загрузи два фото в Space UI
2. Нажми кнопку "Run" или "Submit"

## Шаг 3: Найди запрос
1. В Network найди запрос, который ушёл при нажатии кнопки
2. Обычно это `/queue/data` или `/run/predict`
3. Кликни на запрос

## Шаг 4: Скопируй данные
1. Вкладка **Headers**:
   - Скопируй **Request URL** (полный URL)
   - Скопируй **Request Method** (POST)
   - Скопируй все **Request Headers**

2. Вкладка **Payload** (или **Request**):
   - Скопируй весь JSON, который отправляется
   - Обрати внимание на структуру `data` массива

3. Вкладка **Response**:
   - Посмотри, что возвращается
   - Обрати внимание на формат ответа

## Шаг 5: Пришли мне
Пришли скриншоты или скопированный текст из:
- Request URL
- Request Headers
- Payload (JSON)
- Response (JSON)

Я адаптирую наш код под точный формат, который использует UI.
